05 / CONTENT POLICY
投稿すべての基本方針

どの投稿を作るときも、この5つの方針に照らし合わせる。「これを守っているか？」をチェックリストとして使う。

1
「共感」から入って「行動」で終わる
PRINCIPLE
ターゲットの感情に先に寄り添い、「そうそう、それ私だ」という共感を最初の5秒で作る。その上で、行動への誘導（CTA）で締める。共感なき行動誘導は売り込みにしか聞こえない。
EXAMPLE
❌「転職エージェントに登録しよう」
✅「職場が嫌で眠れない夜、星座はあなたに何を伝えているか」→ 最後に「詳しくはプロフへ」
2
「決断させる」ではなく「許可を与える」
PRINCIPLE
占いの習慣的利用者の46.9%が「決断の後押し」を期待している。ターゲットはすでに転職を「したい」と思っている。「転職しなさい」ではなく「転職を考えていいんだよ」という許可を与えることが最大の価値提供。
EXAMPLE
❌「今すぐ転職した方がいい理由3つ」
✅「仕事が辛いと感じたら、それは星があなたに変化を促しているサインかもしれない」
3
占いは「エンタメ」として使う。断言しない
PRINCIPLE
利用者の多くが「いい結果の時だけ信じる」感覚で占いを見ている。断言・保証的な表現は信頼を損なう。「〜かもしれない」「〜のサインです」という柔らかい表現で、エンタメと実用の境界線を保つ。
EXAMPLE
❌「牡羊座は今月必ず転職成功する」
✅「牡羊座、今月は金星の影響で転職運が高まっているかもしれません」
4
「星座占い」を軸に、タロット・四柱推命は補助
PRINCIPLE
占いの種類として「結果を参考にする」割合が最も高いのは「星座占い・誕生日占い（各5〜6割）」。タロット・四柱推命は30代以降の女性に人気だが、コンテンツの軸は誰でも知っている「星座」に絞ることで最大リーチを確保。
PRIORITY

サブ：タロット（将来の選択系）
深堀：四柱推命（有料鑑定向け）

06 / HOOK FORMULA
最初の3秒で刺さるフック設計

TikTokのアルゴリズムは「視聴完了率」を最重視する。最初の3秒でスクロールを止めるフック文の型を固定化する。

BUZZ型
【ランキング型】「◯◯な星座TOP3」
「転職に向いてる星座TOP3発表🌟」
自分の星座が何位か気になり最後まで見る。コメント欄に「私は何位でした！」が集まりアルゴリズム評価が上がる。
BUZZ型
【警告型】「◯◯してる人は注意」
「今の職場を離れるべき運勢サイン3選⚠️」
「これ、私のことかも」という自己投影が視聴継続を促す。「3選」で最後まで見させる型。
信頼型
【共感型】「◯◯で眠れない人へ」
「仕事のことが頭から離れなくて眠れない人へ、星からのメッセージ🌙」
感情状態を正確に言語化することで「この人は私のことをわかっている」という信頼が生まれる。
信頼型
【疑問型】「なぜ◯◯なのか、星座で読み解く」
「なぜあなたは転職できないのか、星座が答えを知っている」
「なぜ」という問いが好奇心を刺激。答えを知りたくて最後まで視聴する。
CTA型
【体験型】「私が実際に◯◯してみた」
「星座診断に従って転職エージェントに登録してみた正直な感想」
体験談形式は信頼性が高くアフィリへの誘導が自然に行える。「実際に使った」という一人称が説得力を生む。
07 / CONTENT NG LIST
やってはいけない表現・方針

法令・プラットフォーム規約・ターゲット心理のいずれかに反する表現。データが示すターゲットの感度から逆算して設定する。

NG表現・行動なぜNGか代替案
NG 「絶対に転職すべき」「100%成功する」 景品表示法・断定的判断の提供にあたる可能性。信頼損失リスク大 OK 「転職を考えていいかもしれない」
NG 毎回違うアフィリサービスを紹介する 一貫性がなく「宣伝アカウント」に見えてフォロワーが離れる OK 最初はdoda・マイナビ転職の2つに絞る
NG 医療・法律・財務に踏み込む鑑定 資格なしでの診断行為にあたり法的リスク。アカウント停止の可能性 OK 「キャリアの運勢」の範囲に留める
NG 「今すぐ登録して」を毎回言う 押し売り感でフォロー解除・ブロックが増える。全体CVRが下がる OK CTA型は全投稿の15%以内に留める
NG 特定の転職先・企業を批判する 名誉毀損・炎上リスク。プラットフォーム規約違反になる可能性 OK 「あなたの星座に合う環境」という表現に
NG AIが生成したコンテンツであるかのような誤認誘導（AIラベル未表示） TikTok・Instagramは2024年以降AI生成コンテンツへのラベル表示を義務化 OK AI生成動画にはラベルを必ず付ける
