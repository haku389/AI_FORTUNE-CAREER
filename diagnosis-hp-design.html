01 / USER FLOW
5ステップの診断体験設計

SNSから流入したユーザーが「診断完了→アフィリCV→シェア」するまでの完全な動線。各ステップの目的と設計意図を定義する。

STEP 01 — LANDING
🌙 トップページ（ランディング）
SNS動画の「プロフのリンクから」で流入するユーザーの最初の画面。3秒で「これは自分のための診断だ」と感じさせる。
  • ◈キャッチコピー：「あなたの転職タイミング、星が知っている」
  • ◈サブコピー：「生年月日と3つの質問で、星座×転職タイプを診断。今すぐ無料で試す」
  • ◈実績表示：「累計◯万人が診断」（数が増えたら表示）
  • ◈CTAボタン：「無料で診断スタート →」ボタン1つだけ
  • ◈デザイン：星空・神秘的な紫〜紺グラデーション。スマホ縦型最適化
STEP 02 — INPUT
📅 生年月日入力（星座自動判定）
生年月日を入力すると星座を自動判定。「あなたは◯◯座です」と表示することで、診断への没入感を高める。
  • ◈入力形式：年・月・日のセレクトボックス（スマホで打ちやすい）
  • ◈判定演出：星座アイコンがアニメで出現する
  • ◈プライバシー：「個人情報は取得しません」の一文を表示
STEP 03 — QUESTIONS
❓ 3問の仕事診断
転職タイプを判定するための3問。1問につき4択。UIをゲーム的に設計して離脱を防ぐ。
Q1
今の職場への満足度は？
→ 緊急度判定
もう限界に近い 不満はあるが続けられる まあまあ満足 特に問題ない
SCORE AXIS: 転職緊急度（高←→低）
Q2
仕事で一番大事にしたいことは？
→ 転職タイプ判定
年収・キャリアアップ 職場環境・人間関係 やりがい・天職 安定・ワークライフ
SCORE AXIS: タイプ分類（A/B/C/D）
Q3
転職の経験は？
→ サービス推薦精度向上
未経験（初めて） 1回ある 2〜3回ある 4回以上ある
SCORE AXIS: サービスマッチング（初心者向け←→ハイキャリア）
STEP 04 — RESULT
🌟 診断結果ページ（最重要）
ここが収益化の核心。ユーザーが「自分専用の結果をもらった」と感じる設計。アフィリリンクは「おすすめサービス」として自然に配置。
  • ◈星座 + 転職タイプ名を大きく表示（例：「牡羊座の環境改善型」）
  • ◈転職運スコア（0〜100）をアニメーションで表示
  • ◈タイミング診断（今すぐ / 3ヶ月以内 / 半年後 / まだ早い）
  • ◈Claude APIが生成した「あなただけの鑑定文」（50〜100文字）
  • ◈「あなたの星座×タイプに合うサービス」として2〜3件を推薦 →アフィリリンク
  • ◈「もっと詳しく占ってもらう」→LINE登録へ（個別鑑定の入口）
STEP 05 — SHARE
📤 シェア → バイラルエンジン
結果ページの下部に常設。シェアした人の診断URLが拡散 → 新規ユーザーが流入 → 診断 → アフィリCV というループを作る。
  • ◈OGP画像自動生成シェア時に星座・タイプ・スコアが入った画像が自動生成される
  • ◈TikTok / X / Instagram / LINE 4媒体に対応したシェアボタン
  • ◈シェアテキスト自動生成：「私は◯◯座の△△型！転職運スコア82点🔮 あなたも診断して → [URL]」
  • ◈URLパラメータで「どの投稿から来たか」を追跡可能（アナリティクス）

診断HPの収益化メカニズム

06 / TECH STACK
実装に必要な技術と構成

エンジニア経験がある場合の推奨構成。React Nativeの経験があるため、Next.js（React）での実装が最短経路。

Next.js
フロントエンド
React Native知識を活かせる。OGP画像生成（og:image）、SSR、アナリティクス統合がすぐできる。
無料
Vercel
ホスティング
Next.jsとの相性が最高。無料プランで月10万PV以上捌ける。独自ドメイン対応。デプロイが1コマンド。
無料（Hobbyプラン）
Claude API
AI鑑定文生成
診断結果の「あなただけの鑑定文」をリアルタイム生成。星座×タイプ×Q1スコアをプロンプトに入れてパーソナル化。
〜$0.02/件（月数百円〜）
Vercel OG
シェア画像生成
@vercel/ogライブラリで診断結果のシェア用OGP画像を動的生成。星座・スコア・タイプを入れた画像が自動で作られる。
無料（Vercel込み）
Supabase
データ計測
診断数・シェア数・アフィリクリック数を記録。フィットネスアプリで使用経験があるため導入がスムーズ。
無料（Freeプラン）
