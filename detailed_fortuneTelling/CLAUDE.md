## プロジェクト概要

- **プロダクト名**: 精密占い（転職運命精密鑑定）
- **運営キャラクター**: AI占い師◇ルナ（@hoshiyomi_luna）
- **目的**: 簡易診断では得られない「その人固有の分析」を提供する有料コンテンツ
- **価格帯（想定）**: ¥480〜¥980（Stripe決済）
- **前提**: 簡易診断完了後に「もっと詳しく知りたい方へ」として誘導
- **ターゲット**: 28〜38歳の転職を迷う女性

## 簡易診断との違い

| 比較点 | 簡易診断 | 精密占い |
|--------|---------|---------|
| 質問数 | 生年月日＋5問 | 生年月日＋23問（6ブロック） |
| 星の使い方 | 太陽星座のみ | 太陽星座＋月星座＋本命星 |
| 性格分析 | なし | MBTI連携で深い性格把握 |
| 業界・職種 | 4タイプ大分類のみ | 具体的な職種・業界名で絞り込み |
| 時間軸 | タイミング判定のみ | 3ヶ月の月別アドバイス付き |
| 価格 | 無料 | ¥480〜¥980 |

## 課金フロー設計

```
簡易診断の結果ページ（下部）
  └─ 「もっと詳しく知りたい方へ」バナー
       └─ 「精密占いを受ける」ボタン
            └─ 入力フォーム（ニックネーム + 生年月日 + オプション出生時間）
                 └─ 23問診断フロー（6ブロック）
                      └─ Stripe決済画面（¥480〜¥980）
                           └─ 決済完了 → 結果ページ表示
```

### UXポイント
- 簡易結果の「続きが気になる」感情が最高潮の時点で課金導線を出す
- 「¥480で本格鑑定」という低単価設定で離脱を防ぐ
- Stripeのワンクリック決済（保存カード）を活用
- 決済完了後は即座に結果表示（待ち時間ゼロ）

### 本命星（九星気学）

```typescript
// lib/honmeiStar.ts
// 1月・2月生まれは前年度で計算（節分〈2月3日〉基準）
const HONMEI_STARS = [
  '一白水星','二黒土星','三碧木星','四緑木星','五黄土星',
  '六白金星','七赤金星','八白土星','九紫火星'
];

export function getHonmeiStar(year: number, month: number): string {
  const adjustedYear = (month <= 2) ? year - 1 : year;
  const idx = (10 - (adjustedYear % 9)) % 9;
  return HONMEI_STARS[idx];
}
```

## 実装上の注意点

1. **簡易診断との統一**: デザイン・コンポーネントは簡易診断から最大限流用する
2. **月画像**: `mix-blend-mode: screen` を適用。`border-radius` は使わない
3. **ローディング演出**: CSSボーダースピナー（SVGは線アーティファクトが出るため禁止）
4. **ランディング**: `height: 100dvh` + `overflow: hidden`（スクロール禁止）
5. **アフィリ表示**: 「※ アフィリエイト広告を含みます」の表示必須（ステマ規制対応）
6. **API呼び出し**: fire-and-forget（失敗してもUXに影響させない）
7. **Stripe**: PaymentIntent作成はサーバーサイド（`/api/payment/route.ts`）で行う。クライアントに secret key を渡さない
8. **決済完了確認**: `payment_intent_client_secret` をURLクエリで受け取り、Stripeのconfirm後に結果ページへ遷移
